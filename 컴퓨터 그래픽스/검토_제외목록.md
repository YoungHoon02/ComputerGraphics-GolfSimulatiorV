# 코드 검토 — 제외 목록 (AI가 읽으면 깨지는 파일)

> 목적: `/loop` 검토 시 아래 파일/폴더는 **읽지도 검사하지도 말 것**.
> 전부 바이너리이거나 Unity/서드파티 자동생성 파일이라, AI가 읽으면 컨텍스트가 깨짐.
> **코드 수정 금지 — 읽기/분석 전용 검토.**

## 1. 통째로 건너뛸 폴더 (자동생성·서드파티)

| 폴더 | 이유 |
|------|------|
| `Library/` | Unity 임포트 캐시 (자동생성) |
| `Temp/` | 빌드 임시 파일 |
| `Logs/` | 로그 |
| `obj/`, `Build/`, `Builds/` | 빌드 산출물 |
| `UserSettings/` | 개인 에디터 설정 |
| `Packages/` (PackageCache 포함) | Unity 패키지 매니저 캐시 |
| `Assets/BOXOPHOBIC/` | 서드파티 에셋 (스카이박스) |
| `Assets/TextMesh Pro/` | Unity 기본 패키지 |
| `Assets/Casual Game Sounds U6/` | 서드파티 사운드 |
| `Assets/Game Buffs/` | 서드파티 에셋 |
